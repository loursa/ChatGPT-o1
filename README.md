## 基础介绍

ChatGPT 提供两种使用方式：
- **官方版本**：需要科学上网，支持完整功能
- **镜像版本**：国内直接访问，中文优化，具备核心功能

|特性|官方版本|镜像版本|
|---|---|---|
|访问方式|需科学上网|直接访问|
|响应速度|取决于网络|较快|
|功能完整度|100%|80-90%|
|中文体验|一般|优化更好|
|费用|固定收费|部分免费|

### 安全建议
